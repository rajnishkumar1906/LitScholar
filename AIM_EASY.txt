Project Name: BookBuddy (Easy Version)

Goal:
Build a simple local book recommendation system that understands user text queries and suggests relevant books.

What this version does:
- Accepts a text query from the user
- Finds semantically similar books using text embeddings
- Returns the top 5 relevant books
- Uses an LLM to explain recommendations

Technologies used:
- FastAPI
- Chroma (text embeddings only)
- Supabase (book metadata)
- Ollama (text-only reasoning)

What this version does NOT include:
- Images or book covers
- OCR or computer vision
- Personalization or user memory
- Agents or complex orchestration

Success criteria:
- One working API endpoint
- Reasonable book recommendations
- Simple explanations generated locally

Purpose:
Finish something small, build confidence, and establish a working foundation.
